# Cosmocraft-the-game!!
  
Cosmocraft is an exploration game where player can travel through space, land on planets. They can mine, build, and craft of any planet according to their resources. 

# Why making Cosmocraft ?

- Inspired from minecraft, cosmocraft was made to wnble an experience of exploration, building and crafting out side the panetg earth. 

- It has the correct logic and physics that each planet has, like the gravity on each planet is different, giving a unique experience to players.

- I made this game a i wanted to learn a very important toll called THREE.js, which is used to make 3d animations and elements on web.
